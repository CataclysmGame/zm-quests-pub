# Consensys Linea Quests Contracts

This project contains the smart contracts of Cataclysm quests for Linea Voyage.

## Getting Started

Install the required dependencies

```shell
npm i
```

## Configuration

Create a .env file containing the following keys:

```
SIGNER=
METADATA_CID=
CONTRACT_ADDRESS=
ARCADE_THRESHOLD=
ENDLESS_THRESHOLD=
PRIVATE_KEY=
RPC_URL=
PINATA_KEY=
PINATA_SECRET=
```
